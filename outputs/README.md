## 3. 対象＝Solafune の投資判断サマリ（詳細は01・02）

- **何者か**：衛星データ×AIの「Planetary Intelligence OS」（GEOINT＋OSINT＋SIGINT統合）。防衛・安保を中核に、重要鉱物探査・モニタリング（SHIGEN AI）へ応用。2020年設立・東京・未上場。2022年シード2億／2026年3月シリーズA 50億円超（Globisリード＋メガバンクのデット、**バリュエーション非開示**）。
- **なぜ投資に値するか**：センサー非依存の解析レイヤー（ハードを持たず資本効率高）＋120カ国コミュニティ＋政府/国際機関リレーション＋経済安保の政策追い風（防衛費8.8兆・宇宙戦略基金1兆・日米重要鉱物アクションプラン）。VC競争密度が低くリードで持分を厚く取れる＝当ファンドのテーゼ適合度が高い。
- **定量（仮・要検証／02財務モデル）**：
  - 売上 2026=4→2027=10→2028=22→2029=45→**2030=85億円**→2032=101億円。**SOMボトムアップ積算**で裏付け（A日本政府30＋B宇宙基金15＋C同志国MoU6＋D経済安保14＋E民間20＝85億/38件→101億/46件、01§2-3）。Exit逆算の必要売上 年約88億とSOMが一致＝「市場のX%」でなく「誰にいくらで何件」。
  - グロスマージン **60%→72%**（ソフト/解析経済性）
  - 投資時点 **投資前Pre 31億円**（旧40億から引下げ交渉）→投資後Post 36.5億、初回5.5億→**エントリー持分 5.5/36.5≒15.1%**（教材「持分＝チケット/Post」準拠。持分はチケット増ではなくPre交渉で確保。下方ケース＝Pre40億据置なら投資後45.5億で約12%でも検算は成立）
  - **Exit 2032年**：Exit売上 約101億 × EV/Sales **17倍（保守15〜18倍）** → **Exit企業価値 約1,717億円**。倍率は**comps採用群（Planet6/Synspective7/BlackSky10/Anduril14/Palantirプレ20）の中央値10倍**を起点にソフト/主権プレミアムで17倍へ統計的導出、**Palantir現況60.7倍はバブルとして除外**（01§5-1）。
  - Exit持分 **12%** → ポジション **約206億円**（**総コミット16.5億→MOIC約12.5倍**）＝ **ファンド倍率 約1.37X**（感度：Exit持分10%以上 × 15倍以上で150億成立）
  - **ポートフォリオ（べき乗則）**：02の「ポートフォリオ」シート参照。Solafune（FR・206億）＋勝ち馬B（7-8X）＋3-5X×2＋1-2X×3＋0-1X×3の10社で、**投下約123億→グロス約524億＝4.3X→ネットDPI約3.0X**（上位2社で回収の約65%）。Solafune以外9社は分布の"形"を示す例示（仮）。＝「1社1.37X」と「ファンド3.0X」は別レイヤー（中核1社の証明 vs 分布の合算）。
  - Exitパス：①東証グロースIPO（主）②国内重工プライム/商社M&A（代替）③セカンダリ（企業価値×(1−割引30%)）
- **最大の弱点（QAで必ず突かれる／04参照）**：(a) 実シード2億 vs 想定5.5億の不整合（→単独リードでシード〜プレA拡大組成で再設計）、(b) EV/Sales倍率の持続性（防衛AIバブル懸念）、(c) Post-Val交渉（31億まで下げて持分15%）の蓋然性、(d) 政府依存・調達単年度リスク、(e) 1社集中。
